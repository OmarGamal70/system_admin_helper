# system_admin_helper
 System-Admin-Helper is a GUI that help Linux admin to manage there users and groups
 #The admin could do the following through the GUI :
 - Add user
 - Modify user
 - Delete user
 - List users
 - Add Group
 - Modify Group
 - Delete Group
 - List Groups
